Core Features

Project-Based Download provides a complete organized download workflow

Pre-Download Scanning

Scan your Grok favorites before downloading. See total counts for images, SD videos, and HD videos with detailed breakdowns.

Smart Auto-Grouping

Automatically organizes files into project folders by creation date and generation prompt. No manual sorting needed.

Metadata JSON Export

Each project includes a metadata.json file with generation prompts, timestamps, and file details for future reference.

Download Statistics

Track download progress with real-time stats. See completed, failed, and remaining items at a glance.

How Project Download Works

Four simple steps to organize and download your entire Grok library

1

Scan Your Media

Click 'Scan' to analyze your Grok favorites. The extension counts all images and videos available for download.

2

Preview & Configure

Review the scan results showing image count, video count, and estimated download size. Choose what to include.

3

Create Project

Give your project a name. Files will be auto-organized into folders by date (e.g., 2024-03-15/) with prompt-based subfolders.

4

Download Everything

Start the batch download. Files are saved in organized project folders with metadata.json for each group.

Technical Highlights

Hierarchical Folder Structure

Creates nested folders: Project Name / Date / Prompt Group, keeping thousands of files perfectly organized.

Metadata Preservation

Every download group includes a JSON file with the original Grok prompt, timestamps, and file checksums.

Concurrent Downloads

Uses Chrome's download API with smart concurrency control to maximize speed without overwhelming the browser.

Resume Support

If a download is interrupted, the extension tracks progress and can resume from where it left off.
