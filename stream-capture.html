Core Features

Stream Capture provides a seamless real-time image collection workflow

Real-time DOM Observer

Uses MutationObserver to watch for new images as Grok generates them. Detection happens instantly, no manual refresh needed.

Auto Queue System

Every detected image is automatically added to the download queue. Just keep scrolling — Stream Capture handles the rest.

One-Click Batch Download

When you're done browsing, download all captured images at once. No more saving one by one.

Smart Deduplication

Automatically filters out duplicate images already in your collection using hash comparison.

How Stream Capture Works

Four simple steps to capture every Grok image automatically

1

Start Monitoring

Open the extension and click 'Start Stream Capture' to begin real-time monitoring of the Grok page.

2

Browse & Scroll

Browse Grok normally. Every new image that appears on the page is automatically detected and queued.

3

Review Captured

Check the capture queue to see all auto-captured images. Remove any you don't want.

4

Batch Download

Click 'Download All' to save every captured image to your local drive in one go.

Technical Highlights

MutationObserver API

Uses the browser's native MutationObserver to efficiently detect DOM changes without performance impact.

Background Service Worker

Runs in Chrome's service worker for persistent monitoring even when switching tabs.

IndexedDB Cache

Stores captured image metadata in IndexedDB for fast dedup checking and queue management.

Privacy First

All image detection and downloading happens locally. No data is sent to any external server.
