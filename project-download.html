Frequently Asked Questions

Everything you need to know about Project Download

What is Project Download?

Project Download is a PRO feature that lets you download all media from an entire Grok project in one operation. A Grok "project" is a collection of related conversations, generated images, and videos grouped under a single project in your Grok workspace. Instead of downloading items individually, Project Download fetches everything at once and organizes it locally.

What content is included in a project download?

A project download includes all images and videos generated within that project. Files are organized by conversation or generation batch so the structure of your Grok project is reflected in the downloaded folder.

How do I start a project download?

Navigate to your Grok project page, open the GrokMediaDownloader extension, and click "Download Project." The extension scans the project for all media assets, presents you with a summary, and begins downloading once you confirm.

Does Project Download preserve the original file names?

Yes. Project Download retains the original file names assigned by Grok wherever available. When original names are not available, files are named systematically using the project name, generation date, and a sequential index to ensure each file is uniquely identifiable.

What happens if a download is interrupted?

Project Download tracks progress locally. If a download is interrupted — due to a network error or browser closing — you can resume it by reopening the extension on the same project page. Already-downloaded files are skipped, so only the remaining files are fetched.

Can I download multiple Grok projects at once?

Currently, Project Download processes one project at a time. You can start a download for one project, then switch to another project page and start another download. Each download runs independently and does not block the other.
