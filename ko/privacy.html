개인정보처리방침

최종 업데이트: 2025년 12월

1. 데이터 수집 및 사용 GrokMediaDownloader는 완전히 클라이언트 측에서 작동합니다. 개인 데이터, 브라우징 기록, 다운로드 콘텐츠를 수집, 저장, 공유하지 않습니다. 모든 처리는 브라우저 내에서 로컬로 수행됩니다. 2. 권한에 대하여 요청하는 권한은 핵심 기능 실행에만 사용됩니다: - Grok 페이지의 미디어 감지 - 미디어 파일을 컴퓨터에 저장 - 설정의 로컬 저장 3. 제3자 서비스 결제는 Lemon Squeezy에서 처리합니다. 재무 데이터는 Lemon Squeezy가 직접 처리 및 보호하며, 본 확장 프로그램은 민감한 재무 정보에 접근하지 않습니다. 4. 정책 업데이트 본 정책을 수시로 변경할 권리를 보유합니다. 변경 사항은 본 페이지에서 업데이트됩니다.

2. 권한 설명

당사가 요청하는 권한("Storage", "Downloads", "Host Permissions" 등)은 확장 프로그램의 핵심 기능을 실행하기 위해서만 사용됩니다:

  • 현재 탐색 중인 Grok 페이지를 읽어 미디어를 감지합니다.
  • 지정한 위치에 파일을 다운로드합니다.
  • 확장 프로그램의 로컬 설정과 다운로드 기록을 저장합니다.

3. 제3자 서비스

본 확장 프로그램은 결제 처리에 LemonSqueezy를 사용합니다. 라이선스 구매 시 고객의 결제 정보는 LemonSqueezy의 개인정보처리방침에 따라 처리됩니다. 당사는 라이선스 인증에 필요한 최소한의 정보(라이선스 키 등)만 수신합니다.

4. 데이터 보안

당사는 어떠한 데이터도 수집하지 않으므로 데이터 유출 위험이 없습니다. 고객의 모든 콘텐츠는 고객의 기기에 보관됩니다.

5. 정책 업데이트

중요한 변경 사항이 있을 경우 확장 프로그램 업데이트를 통해 사용자에게 알립니다. 서비스의 지속적인 사용은 업데이트된 정책에 대한 동의를 의미합니다.
