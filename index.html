PRO 級功能

基於真實使用者反饋，我們構建了最強大的 Grok 媒體管理系統。

智慧清單生成 (Generate List)

無需手動逐一尋找。Extension 會自動掃描頁面與對話紀錄，智慧生成所有可下載的媒體清單，讓您預覽後再決定。

日期與歷程過濾 (PRO)

只想下載今天的作品？使用 Date Filter 指定日期範圍。系統還會自動比對 Download History，避免重複下載相同的檔案。

數據管理與備份 (PRO)

更換電腦也不怕遺失紀錄。透過 Export/Import Data 功能，您可以輕鬆備份或轉移您的下載歷程資料庫。

批量清理 (Bulk Remove)

下載完成後想清理空間？Bulk Remove 讓您能依照日期範圍，批量移除 Grok 上已收藏的舊圖片與影片，保持媒體庫整潔。

簡單三步驟，自動化您的下載

01

設定過濾條件

開啟 Extension 面板，選擇您要下載的媒體類型（Images/Videos）。若是 PRO 用戶，可設定 "Only after" 日期過濾器。

02

1. Generate List

點擊藍色按鈕。Extension 會快速掃描 Grok 頁面並與您的歷史紀錄比對，列出所有待下載的新檔案。

03

2. Start Download

點擊綠色按鈕。放鬆休息，看著進度條跑動，檔案將自動分類儲存至您的 grok_downloads/ 資料夾。
