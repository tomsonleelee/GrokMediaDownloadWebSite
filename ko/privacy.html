개인정보처리방침

최종 업데이트: 2026년 4월

본 개인정보처리방침은 Kario LLC(이하 "당사")가 GrokMediaDownloader Chrome 확장 프로그램(이하 "본 확장 프로그램") 및 마케팅 웹사이트 grokmedia.kario-studio.com(이하 "본 웹사이트")과 관련된 정보를 어떻게 처리하는지 설명합니다. 당사는 고객의 개인정보를 보호하고 당사의 관행에 대해 투명성을 유지하기 위해 최선을 다하고 있습니다.

1. 수집하는 정보

확장 프로그램 내: GrokMediaDownloader는 100% 클라이언트 측에서 작동하는 도구입니다. 당사는 외부 서버에서 개인 데이터를 수집, 전송 또는 저장하지 않습니다. 본 확장 프로그램은 고객의 이름, 이메일 주소 또는 기타 개인 식별 정보를 수집하지 않습니다. 확장 프로그램 내부에는 분석, 텔레메트리 또는 추적 코드가 일절 포함되어 있지 않습니다. 다운로드, FFmpeg WebAssembly를 통한 비디오 병합, 스트림 캡처를 포함한 모든 미디어 처리는 고객의 브라우저에서 로컬로 완전히 수행됩니다.

웹사이트에서: 당사의 마케팅 웹사이트는 페이지 조회수, 유입 경로, 일반적인 사용 패턴 등 표준 웹사이트 트래픽 분석을 위해 Google Analytics 4(GA4)를 사용합니다. GA4는 익명화된 집계 데이터를 수집하기 위해 쿠키를 사용할 수 있습니다. 이 분석 데이터는 확장 프로그램과 완전히 독립적이며 개별 사용자를 식별하지 않습니다. GA4의 표준 구성에서 제공하는 데이터 외에 본 웹사이트를 통해 개인 정보를 수집하지 않습니다.

2. 정보 이용 방법

확장 프로그램은 어떠한 데이터도 수집하지 않으므로 사용, 공유 또는 판매할 사용자 데이터가 없습니다. 확장 프로그램 내에서 모든 작업은 로컬로 수행되며 당사의 서버나 제3자 서버로 정보가 전송되지 않습니다.

마케팅 웹사이트의 익명화된 집계 분석 데이터는 일반적인 트래픽 패턴을 파악하고 웹사이트 경험을 개선하는 데에만 사용됩니다. 당사는 이 데이터를 사용하여 개별 사용자를 식별, 연락 또는 타겟팅하지 않습니다.

3. 데이터 저장

확장 프로그램에 의해 생성되는 모든 데이터 -- 다운로드 기록, 대기열 데이터, 사용자 설정, 임시 비디오 파일 포함 -- 는 Chrome의 내장 스토리지 API(chrome.storage.local) 및 IndexedDB를 사용하여 고객의 로컬 기기에 저장됩니다. 이 데이터는 고객의 기기를 떠나지 않으며 당사나 제3자가 접근할 수 없습니다.

고객은 언제든지 로컬 데이터를 완전히 제어할 수 있습니다. 확장 프로그램의 내장 데이터 관리 기능을 사용하여 데이터를 내보내기, 가져오기 또는 삭제할 수 있습니다. 확장 프로그램을 제거하면 모든 로컬 데이터도 자동으로 삭제됩니다.

4. 제3자 서비스

결제 처리: PRO 라이선스 구매는 제3자 결제 처리업체인 LemonSqueezy에서 처리됩니다. 라이선스를 구매할 때 고객의 결제 정보(신용카드 정보 등)는 LemonSqueezy의 자체 개인정보처리방침에 따라 LemonSqueezy가 직접 수집하고 처리합니다. Kario LLC는 고객의 신용카드 정보에 직접 접근, 저장 또는 처리하지 않습니다. 당사는 라이선스 인증에 필요한 최소한의 정보(라이선스 키 및 주문 상태 등)만 수신합니다.

Google Analytics: 마케팅 웹사이트(확장 프로그램 아님)는 트래픽 분석을 위해 Google Analytics 4를 사용합니다. Google Analytics 차단 브라우저 부가 기능을 사용하거나 브라우저의 쿠키 설정을 조정하여 Google Analytics를 비활성화할 수 있습니다.

5. Chrome 확장 프로그램 권한

본 확장 프로그램은 핵심 기능에 필요한 최소한의 권한만 요청합니다. 각 권한의 용도는 다음과 같습니다:

  • activeTab -- 현재 보고 있는 Grok 페이지와 상호작용하여 사용 가능한 미디어를 감지하고 목록을 표시합니다.
  • storage -- 다운로드 기록, 확장 프로그램 설정, 대기열 데이터를 로컬 기기에 저장합니다.
  • downloads -- 미디어 파일을 컴퓨터의 로컬 파일 시스템에 저장합니다.
  • sidePanel -- Chrome 사이드 패널에 표시되는 Video Studio 및 대기열 사용자 인터페이스를 구동합니다.
  • offscreen -- 성능 향상을 위해 FFmpeg WebAssembly 비디오 처리를 백그라운드 문서에서 실행합니다.
  • 호스트 권한(*://grok.com/*) -- 확장 프로그램의 작동을 grok.com 도메인으로만 제한합니다. 본 확장 프로그램은 다른 웹사이트에서 실행되거나 다른 웹사이트에 접근하지 않습니다.

본 확장 프로그램은 쿠키를 설정하지 않습니다. 이러한 권한을 통해 수집된 데이터는 외부로 전송되지 않습니다.

6. 데이터 보안

확장 프로그램은 고객의 로컬 기기에서 완전히 작동하며 외부 서버로 데이터를 전송하지 않으므로, 확장 프로그램을 통한 데이터 유출 위험은 사실상 제거됩니다. 고객의 미디어 파일, 다운로드 기록, 설정은 고객의 기기에만 보관되며 고객이 직접 관리합니다. 당사는 확장 프로그램의 사용자 데이터를 저장하는 서버를 운영하지 않습니다.

7. 아동 개인정보

본 확장 프로그램은 만 13세 미만의 아동을 대상으로 하지 않습니다. 당사는 아동의 개인정보를 의도적으로 수집하지 않습니다. 본 확장 프로그램은 어떤 사용자로부터도 개인정보를 수집하지 않으므로, 미성년자의 데이터를 부주의하게 수집할 위험이 없습니다.

8. 정책 변경

당사는 관행의 변경을 반영하거나 법적 및 규제적 이유로 본 개인정보처리방침을 수시로 업데이트할 수 있습니다. 중요한 변경을 할 경우 본 페이지 상단의 "최종 업데이트" 날짜를 변경하고 확장 프로그램 업데이트를 통해 사용자에게 알릴 수 있습니다. 변경 후에도 확장 프로그램 또는 웹사이트를 계속 사용하면 업데이트된 정책에 동의하는 것으로 간주됩니다.
