동영상 스토리 제작 스튜디오

핵심 기능

Story Mode Helper는 완전한 동영상 스토리 제작 도구 체인을 제공합니다

비주얼 Storyboard

드래그 앤 드롭으로 동영상 세그먼트를 직관적으로 관리. Grok 즐겨찾기에서 원클릭으로 추가 가능.

프레임 추출

각 동영상의 첫 번째와 마지막 프레임을 자동 추출하여 썸네일 미리보기로 표시.

프레임을 Grok에 전송

동영상의 마지막 프레임을 선택하고 프롬프트를 추가하여 Grok Imagine에 전송. 후속편 동영상을 생성.

로컬 FFmpeg 병합

WebAssembly 버전 FFmpeg로 브라우저 내에서 로컬로 동영상을 병합. 클라우드 업로드 불필요, 완전한 프라이버시 보호.

Story Mode 사용 방법

네 가지 간단한 단계로 전문 동영상 스토리를 만드세요

1

동영상을 스튜디오에 추가

Grok 즐겨찾기 페이지에서 "Add to Studio"를 클릭하거나, 사이드 패널에 로컬 동영상을 드래그 앤 드롭.

2

스토리 편집

Storyboard에서 세그먼트를 드래그 앤 드롭하여 순서를 변경. 첫 번째 동영상이 마스터가 되어 출력 사양을 결정.

3

후속편 생성

세그먼트를 선택하여 마지막 프레임을 표시. 프롬프트를 입력하여 Grok에 전송하고 후속편 동영상을 생성.

4

병합 및 내보내기

모든 세그먼트가 준비되면 내보내기를 클릭. FFmpeg가 로컬에서 동영상을 병합하여 다운로드.

기술 하이라이트

IndexedDB 대용량 스토리지

Chrome Storage 10MB 제한을 극복하고, IndexedDB를 사용하여 대용량 동영상 Blob 데이터를 저장.

WebAssembly FFmpeg

FFmpeg.js (WASM)를 사용하여 브라우저 내에서 전문급 동영상 처리. 서버 불필요.

Chrome Side Panel API

Chrome 최신 Side Panel API를 채택하여 상주형 작업 공간을 제공. 메인 페이지 탐색을 방해하지 않음.

완전한 로컬 처리

모든 동영상 데이터는 로컬에서 처리되며, 어떤 서버에도 업로드하지 않아 창작물의 프라이버시를 보장.

자주 묻는 질문

스토리 모드에 관한 모든 질문에 답변드립니다

GrokMediaDownloader의 스토리 모드란 무엇인가요?

스토리 모드는 다운로드한 Grok AI 이미지를 스크롤 가능한 비주얼 스토리나 슬라이드쇼로 구성할 수 있게 해줍니다. 단순한 이미지 폴더가 아니라 의미 있는 순서로 배열하고, 전체 흐름을 미리 보고, 전체 스토리를 하나의 파일로 공유하거나 내보낼 수 있습니다. Grok으로 이미지 시리즈나 비주얼 내러티브를 제작하는 크리에이터에게 이상적입니다.

Grok 이미지로 스토리를 어떻게 만드나요?

확장 프로그램에서 스토리 모드 패널을 열고, 컬렉션에서 이미지를 선택한 다음, 드래그 앤 드롭으로 원하는 순서로 배열하고 "스토리 만들기"를 클릭합니다. 스토리가 즉시 생성되어 미리 보기나 내보내기를 할 수 있습니다.

스토리의 이미지 순서를 커스텀할 수 있나요?

네. 스토리 모드에는 드래그 앤 드롭 재정렬 인터페이스가 포함되어 있습니다. 스토리를 완성하기 전에 이미지를 자유롭게 재배열할 수 있습니다. 설정한 순서는 내보낸 결과물에 유지됩니다.

스토리 하나에 이미지를 몇 장까지 넣을 수 있나요?

스토리 모드는 스토리당 수백 장의 이미지를 지원합니다. 최적의 성능을 위해 100장 이내로 유지하는 것을 권장하지만, 확장 프로그램 자체에는 엄격한 기술적 제한이 없습니다.

스토리 모드는 어떤 내보내기 형식을 지원하나요?

스토리 모드는 순서가 있는 이미지의 ZIP 아카이브나 어떤 브라우저에서도 열 수 있는 독립적인 HTML 슬라이드쇼 파일로 내보낼 수 있습니다. 향후 PRO 업데이트에서 더 많은 내보내기 형식이 추가될 수 있습니다.

스토리의 이미지에 캡션이나 제목을 추가할 수 있나요?

네. 스토리 모드에서는 각 이미지에 텍스트 캡션을 추가할 수 있습니다. 캡션은 스토리 뷰에서 이미지 아래에 표시되고 내보낸 HTML 슬라이드쇼에도 포함되어 각 장면에 맥락이나 설명을 추가하기 편합니다.

스토리 모드는 무료인가요, PRO 기능인가요?

스토리 모드는 PRO 전용 기능입니다. $4.99(평생 라이선스)로 GrokMediaDownloader PRO로 업그레이드하면 스토리 모드와 함께 스트림 캡처, Video Gen Queue, 프로젝트 다운로드를 사용할 수 있습니다.
