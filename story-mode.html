核心功能

Story Mode Helper 提供完整的影片故事創作工具鏈

Storyboard 故事板

視覺化拖放排序影片片段，直覺管理您的影片時間軸。支援從 Grok 收藏一鍵添加影片。

畫格擷取

自動擷取每個影片的首尾畫格作為縮圖預覽，讓您快速識別內容。

發送畫格到 Grok

選取影片最後一幀，搭配提示詞一鍵發送到 Grok Imagine，生成續集影片延續故事。

本地 FFmpeg 合併

使用 WebAssembly 版 FFmpeg 在瀏覽器本地合併影片，無需上傳雲端，保護隱私。

Story Mode 使用流程

四個簡單步驟，打造專業影片故事

1

添加影片到工作室

從 Grok 收藏頁面點擊「Add to Studio」按鈕，或直接拖放本地影片到 Side Panel。

2

編排故事順序

在 Storyboard 中拖放影片片段，調整順序。第一個影片會成為 Master，決定輸出規格。

3

擷取畫格生成續集

選擇片段查看最後一幀，輸入提示詞，發送到 Grok 生成續集影片。

4

合併輸出

所有片段準備好後，點擊匯出按鈕，FFmpeg 會在本地合併所有影片並下載。

技術亮點

IndexedDB 大容量儲存

突破 Chrome Storage 10MB 限制，使用 IndexedDB 儲存大型影片 Blob 資料。

WebAssembly FFmpeg

使用 FFmpeg.js (WASM) 在瀏覽器內執行專業級影片處理，無需伺服器。

Chrome Side Panel API

採用 Chrome 最新 Side Panel API，提供常駐式工作區，不干擾主頁面瀏覽。

完全本地處理

所有影片資料在本地處理，絕不上傳至任何伺服器，確保您的創作隱私。
