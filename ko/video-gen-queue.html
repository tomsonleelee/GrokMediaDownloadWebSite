Video Gen Queue

자동 동영상 생성 큐

새로운 사이드 패널 인터페이스로 일괄 동영상 생성이 가능합니다. 4가지 생성 모드, 실패 시 자동 재시도, 스마트 속도 제한 감지로 동영상 생성을 완전 자동화.

전체 워크플로우 데모

큐에 추가 → 설정 → 자동 생성 → 결과 수집

* 동영상은 Video Gen Queue의 전체 워크플로우를 소개합니다

핵심 기능

Video Gen Queue는 완전한 자동 동영상 생성 도구 체인을 제공합니다

🔄

자동 재시도

생성 실패 시 자동으로 큐에 다시 추가하여 모든 작업의 완료를 보장.

📋

일괄 큐 추가

즐겨찾기에서 여러 항목을 선택하여 원클릭으로 일괄 큐에 추가.

🎨

4가지 생성 모드

Custom/Normal/Fun/Spicy 모드로 다양한 크리에이티브 스타일에 대응.

🛡️

속도 제한 감지

Grok API의 속도 제한을 스마트하게 감지하여 자동 일시 중지 및 쿨다운 후 재개.

Video Gen Queue 사용 방법

5단계로 동영상 생성을 자동화

1

Grok Imagine 탐색

grok.com에서 이미지를 탐색 및 생성하고, 동영상으로 만들고 싶은 Posts를 찾으세요.

2

큐에 추가

플로팅 🎯 버튼을 클릭하거나, 일괄 추가에서 시간 범위(1h~72h)를 선택하여 원클릭으로 추가.

3

파라미터 설정

모드(Custom/Normal/Fun/Spicy), 시간(6s/10s), 해상도(480p/720p), 목표 횟수, 재시도 제한을 선택.

4

생성 시작

Start를 클릭. 시스템이 각 Post 페이지로 자동 이동하여 생성 요청을 순차적으로 전송.

5

결과 수집

생성에 성공한 동영상은 Grok 즐겨찾기에 자동으로 표시되며, 언제든 다운로드 가능.

기술 하이라이트

Chrome Side Panel API

Chrome 최신 Side Panel API로 상주형 워크스페이스를 제공. 메인 페이지 탐색을 방해하지 않습니다.

IndexedDB 영구 저장

큐 데이터와 프롬프트 통계를 IndexedDB에 독립적으로 저장. 브라우저를 닫아도 데이터가 사라지지 않습니다.

Fetch Interceptor

MAIN world에서 생성 결과를 인터셉트하며, 90초 타임아웃 폴백 포함. 모든 생성을 정확하게 추적.
