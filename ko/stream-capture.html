스트림 캡처

실시간 Grok 이미지 자동 캡처

브라우징하면서 Grok AI를 모니터링하세요. 스트림 캡처는 DOM 옵저버 기술을 사용하여 실시간으로 새로 생성된 이미지를 자동 감지하고 큐에 추가합니다. 영감을 놓치지 마세요 — 원클릭으로 모든 이미지를 일괄 다운로드하세요.

Chrome에 추가

스트림 캡처 실제 동작 보기

모니터링 시작 → Grok 탐색 → 이미지 자동 캡처 → 일괄 다운로드

* 스트림 캡처 실시간 캡처 워크플로우를 보여주는 동영상입니다

핵심 기능

스트림 캡처는 원활한 실시간 이미지 수집 워크플로우를 제공합니다

실시간 DOM 옵저버

MutationObserver를 사용하여 Grok이 이미지를 생성할 때 새 이미지를 감시합니다. 수동 새로고침 없이 즉시 감지됩니다.

자동 큐 시스템

감지된 모든 이미지가 자동으로 다운로드 큐에 추가됩니다. 계속 스크롤하기만 하세요 — 나머지는 스트림 캡처가 처리합니다.

원클릭 일괄 다운로드

브라우징이 끝나면 캡처된 모든 이미지를 한 번에 다운로드하세요. 더 이상 하나씩 저장할 필요가 없습니다.

스마트 중복 제거

해시 비교를 사용하여 컬렉션에 이미 있는 중복 이미지를 자동으로 필터링합니다.

스트림 캡처 작동 방식

간단한 4단계로 모든 Grok 이미지를 자동 캡처하세요

1

모니터링 시작

확장 프로그램을 열고 '스트림 캡처 시작'을 클릭하여 Grok 페이지의 실시간 모니터링을 시작하세요.

2

탐색 & 스크롤

평소처럼 Grok을 탐색하세요. 페이지에 나타나는 모든 새 이미지가 자동으로 감지되어 큐에 추가됩니다.

3

캡처 항목 확인

캡처 큐를 확인하여 자동 캡처된 모든 이미지를 확인하세요. 원하지 않는 항목은 제거할 수 있습니다.

4

일괄 다운로드

'모두 다운로드'를 클릭하여 캡처된 모든 이미지를 한 번에 로컬 드라이브에 저장하세요.

기술적 특징

MutationObserver API

브라우저의 네이티브 MutationObserver를 사용하여 성능 영향 없이 효율적으로 DOM 변경 사항을 감지합니다.

백그라운드 서비스 워커

Chrome의 서비스 워커에서 실행되어 탭을 전환해도 지속적인 모니터링이 가능합니다.

IndexedDB 캐시

캡처된 이미지 메타데이터를 IndexedDB에 저장하여 빠른 중복 확인 및 큐 관리를 지원합니다.

프라이버시 우선

모든 이미지 감지 및 다운로드는 로컬에서 이루어집니다. 어떠한 데이터도 외부 서버로 전송되지 않습니다.
