核心功能

專案下載提供完整的有組織下載工作流程

下載前預掃描

下載前先掃描您的 Grok 收藏。查看圖片、SD 影片和 HD 影片的總數及詳細分類。

智慧自動分組

按建立日期和生成提示詞自動將檔案整理到專案資料夾中。無需手動排序。

中繼資料 JSON 匯出

每個專案都包含 metadata.json 檔案，記錄生成提示詞、時間戳記和檔案詳細資訊，方便日後參考。

下載統計

透過即時統計追蹤下載進度。一目了然地查看已完成、失敗和剩餘項目。

專案下載運作方式

四個簡單步驟，整理並下載整個 Grok 媒體庫

1

掃描您的媒體

點擊「掃描」分析您的 Grok 收藏。擴充功能會計算所有可供下載的圖片和影片。

2

預覽與設定

查看掃描結果，包含圖片數量、影片數量和預估下載大小。選擇要包含的內容。

3

建立專案

為您的專案命名。檔案會自動按日期整理到資料夾中（例如 2024-03-15/），並建立以提示詞命名的子資料夾。

4

下載所有內容

啟動批次下載。檔案會儲存在有組織的專案資料夾中，每個群組都附帶 metadata.json。

技術亮點

階層式資料夾結構

建立巢狀資料夾：專案名稱 / 日期 / 提示詞群組，讓數千個檔案井然有序。

中繼資料保存

每個下載群組都包含 JSON 檔案，記錄原始 Grok 提示詞、時間戳記和檔案校驗碼。

並行下載

使用 Chrome 的下載 API 搭配智慧並行控制，在不讓瀏覽器過載的情況下最大化速度。

斷點續傳

如果下載中斷，擴充功能會追蹤進度，可以從中斷處繼續下載。

常見問題

關於 Project Download 的所有問題解答

Project Download 是什麼？

Project Download 是一項 PRO 功能，讓你一次下載整個 Grok 專案的所有媒體。Grok「專案」是工作區中一組相關對話、已產生圖片和影片的集合。不需逐一下載，Project Download 會一次取回所有內容並在本機整理好。

專案下載包含哪些內容？

專案下載包含該專案內產生的所有圖片和影片。檔案按對話或產生批次整理，下載的資料夾結構會反映你的 Grok 專案架構。

如何開始專案下載？

前往你的 Grok 專案頁面，開啟 GrokMediaDownloader 擴充功能，點擊「下載專案」。擴充功能會掃描專案中的所有媒體資源，顯示摘要供確認後開始下載。

Project Download 會保留原始檔名嗎？

會。Project Download 盡可能保留 Grok 指定的原始檔名。當原始檔名不可用時，會使用專案名稱、產生日期和序號來系統性命名，確保每個檔案都可以唯一識別。

下載中途被中斷怎麼辦？

Project Download 會在本機追蹤進度。如果因網路錯誤或關閉瀏覽器而中斷，只需在同一個專案頁面重新開啟擴充功能即可繼續。已下載的檔案會被跳過，只會繼續下載剩餘的檔案。

可以同時下載多個 Grok 專案嗎？

目前 Project Download 一次處理一個專案。你可以開始下載一個專案，然後切換到另一個專案頁面再開始另一個下載。每個下載獨立運作，互不干擾。

Project Download 是免費功能嗎？

Project Download 是 PRO 專屬功能。升級至 GrokMediaDownloader PRO（$4.99 終身授權）即可解鎖 Project Download 以及 Stream Capture、Video Gen Queue 和 Story Mode。
