기본 기능 기본 기능

스마트 스캔 기능입니다. Grok/Imagine/Favorites에서 클릭하면 페이지를 로드하고, 모든 미디어를 감지하여 이미 다운로드한 파일을 제외한 리스트를 생성합니다.
PRO 전용 기능입니다. 수천 장이 있을 경우 날짜를 설정하여 '오늘' 또는 '이번 주'의 새 이미지만 다운로드할 수 있습니다.
Generate List는 Favorite List를 스캔합니다. Stream Capture는 지속적으로 모니터링하여 새 이미지를 자동 캡처합니다.
Grok은 저해상도 미리보기를 저장하는 경우가 있습니다. HD Upgrade는 자동으로 감지하여 Grok에 1080p 고화질 버전을 요청합니다. 이미 HD 동영상이 있는 경우에는 중복 요청하지 않습니다. 모든 동영상이 HD 화질인 경우 자동으로 종료됩니다.
기본적으로 이력은 로컬에 저장됩니다. PRO 사용자는 Export/Import Data로 백업과 복원이 가능합니다.

Story Mode Helper Story Mode Helper

v2.0의 주요 기능입니다. 사이드 패널 동영상 스튜디오로, Storyboard 관리, 프레임 추출, Grok 전송, 로컬 병합 출력이 가능합니다.
Story Mode Helper에서 동영상 세그먼트를 선택하여 마지막 프레임을 표시합니다. 프롬프트를 입력하고 전송 버튼을 클릭하면 프레임이 Grok Imagine에 업로드됩니다.
완전히 로컬입니다. WebAssembly FFmpeg를 사용하여 모든 동영상 처리가 브라우저 내에서 완료됩니다. 서버 업로드 없음.
Storyboard의 첫 번째 동영상이 Master로 표시됩니다. 병합 시 다른 동영상은 자동으로 Master의 해상도와 프레임 레이트에 맞춰 조정되어 일관된 출력 품질이 보장됩니다.
IndexedDB를 사용하여 동영상 데이터를 저장합니다. 용량은 브라우저 설정에 따라 일반적으로 수백 MB에서 수 GB까지 저장 가능합니다. 사이드 패널에서 현재 스토리지 사용량을 확인할 수 있습니다.

v2.2 신기능

Video Gen Queue는 v2.2의 신기능입니다. 사이드 패널의 자동 동영상 생성 스케줄링 시스템으로, 여러 동영상 생성 작업을 일괄 추가할 수 있습니다. Custom/Normal/Fun/Spicy 모드를 지원합니다. 무료 버전은 50회, PRO 버전은 무제한입니다.
Project-Based Download는 다운로드 전에 자동 스캔하여 통계 정보(이미지/동영상 수, HD/SD 비율)를 표시하고, 날짜와 프롬프트로 프로젝트 폴더({날짜}_{프롬프트}/)를 자동 생성합니다. PRO 전용 기능입니다.
