### Tag-Based Download

Download media from a specific Grok tag (folder). Select "Download by Tag", pick a tag from dropdown. Files organized under `grok_downloads/tags/{tagName}/`. Separate download history using composite keys so the same file can be downloaded independently from different tags. PRO exclusive.

### Custom Filename Templates

Three-mode filename format selector:
- **Grok Native**: Original format (`grok_image_YYYYMMDD_id`)
- **Prompt Based**: Prompt-based naming (`img_YYYYMMDDHHmmss_prompt`)
- **Custom**: Free-form templates with tokens `{date}`, `{datetime}`, `{id}`, `{shortid}`, `{prompt}`, `{quality}`

Available to all users. Backwards-compatible migration from old `promptAsFilename` setting.

### Frame Lock & Trim-on-Export

Story Mode filmstrip allows selecting any frame (not just last frame) for video continuation. Selected frame is locked (orange indicator) after generation. On export, FFmpeg auto-trims at locked frame to prevent overlap. Enhances existing Frame Extraction and Send Frame to Grok features.
